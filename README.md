# Netflix Data Analysis 📊

## Overview
This project analyzes Netflix movies and TV shows dataset to uncover trends, patterns, and insights about content strategy.

## Key Work Done
- Data Cleaning (handling null values, formatting)
- Feature Engineering (duration split into minutes & seasons)
- Genre Transformation (multi-label to structured format)

## Tech Stack
- Python
- Pandas
- Matplotlib
